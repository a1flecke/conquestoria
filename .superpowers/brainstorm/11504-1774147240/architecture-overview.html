Conquestoria — Architecture Overview

Vanilla TypeScript + Canvas, hosted on GitHub Pages

System Architecture
🎨 Presentation Layer
Canvas Renderer
Hex map, units, fog of war, animations
DOM UI Layer
Menus, panels, tech tree, city view, HUD
Audio Engine
Web Audio API + HTML5 Audio
👆 Input Layer
Touch Handler
Tap, pinch-zoom, pan, long-press
Mouse/Keyboard
Click, hover, shortcuts, scroll
⚙️ Game Engine
Turn Manager
Turn flow, phase resolution
Map System
Hex grid, terrain, resources, fog
City System
Buildings, production, growth
Unit System
Movement, combat, commands
Tech System
Multi-track trees, research
Economy
Resources, trade, markets
Diplomacy
Treaties, vassals, reputation
Espionage
Spy network, missions
Wonder System
Build, quests, bonuses
Threat System
Barbarians, pirates, rebels
🤖 AI Layer
Civ AI
Strategy, expansion, diplomacy decisions
Threat AI
Barbarian/pirate behavior, tribal evolution
💾 Persistence Layer
IndexedDB
Game state, save data
Service Worker
Offline caching, asset management
State Serializer
Save/load, auto-save per turn

This is the high-level system architecture. Each colored layer is independent and communicates through well-defined interfaces. Review in terminal for details.
