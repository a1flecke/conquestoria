# Pike Thrust + Quadruped Gait Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a pike-specific thrust attack animation for `PikemanV2Sprite` and a diagonal-pair 4-leg walk + lunge attack for `ScoutHoundV2Sprite` and `WarHoundV2Sprite`, differentiated by speed/mass via CSS custom properties.

**Architecture:** Introduce a new `data-kind-variant` attribute axis on both the wrapper `<div>` and inner `<svg>` elements. `SpriteFrameV2` emits it automatically when a `variant` prop is supplied. CSS selectors key off `[data-kind-variant="pike"]` for the thrust override and `[data-kind-variant="scout"/"war"]` for hound tempo variables. No game-logic or TypeScript changes — purely JSX source + CSS. Auto-generated `.svg.ts` files are regenerated by re-running the serializer after JSX edits land.

**Tech Stack:** JSX (custom runtime, no React), plain CSS animations, `bash scripts/run-with-mise.sh yarn node scripts/serialize-sprites.mjs` to regenerate serialized SVG bundles.

- [ ] **Step 1: Replace SpriteFrameV2 function definition**

  Open `design/conquestoria-sprites/lib/units-v2.jsx`. Replace lines 42–66:

  ```jsx
  function SpriteFrameV2({
    size = 128, children, hex = true, hexTint = '#000',
    state = 'idle', kind = 'civilian', phase,
  }) {
    const { HexBase } = window;
    const autoPhase = React.useMemo(() => Math.random(), []);
    /* Use strict undefined check: `phase={0}` is a legitimate caller intent
     * (e.g. the phase-desync demo's first column) and must not fall through
     * to autoPhase. Only an *omitted* prop triggers the auto value. */
    const p = phase === undefined ? autoPhase : phase;
    return (
      <div className="cq-sprite-wrap cq-v2" data-state={state} data-kind={kind}
           style={{ '--phase': p }}>
        <svg viewBox={`0 0 ${size} ${size}`} width="100%" height="100%"
             data-state={state} data-kind={kind}>
          {hex && (
            <g transform={`translate(${(size - 96) / 2} ${size - 96 * 0.866 - 6})`}>
              <HexBase size={96} tint={hexTint} />
            </g>
          )}
          <g className="cq-sprite-figure">{children}</g>
        </svg>
      </div>
    );
  }
  ```

  With:

  ```jsx
  function SpriteFrameV2({
    size = 128, children, hex = true, hexTint = '#000',
    state = 'idle', kind = 'civilian', variant, phase,
  }) {
    const { HexBase } = window;
    const autoPhase = React.useMemo(() => Math.random(), []);
    /* Use strict undefined check: `phase={0}` is a legitimate caller intent
     * (e.g. the phase-desync demo's first column) and must not fall through
     * to autoPhase. Only an *omitted* prop triggers the auto value. */
    const p = phase === undefined ? autoPhase : phase;
    const variantAttr = variant ? { 'data-kind-variant': variant } : {};
    return (
      <div className="cq-sprite-wrap cq-v2" data-state={state} data-kind={kind}
           {...variantAttr} style={{ '--phase': p }}>
        <svg viewBox={`0 0 ${size} ${size}`} width="100%" height="100%"
             data-state={state} data-kind={kind} {...variantAttr}>
          {hex && (
            <g transform={`translate(${(size - 96) / 2} ${size - 96 * 0.866 - 6})`}>
              <HexBase size={96} tint={hexTint} />
            </g>
          )}
          <g className="cq-sprite-figure">{children}</g>
        </svg>
      </div>
    );
  }
  ```

### Task 3: Add pike thrust CSS

**Files:**
- Modify: `src/assets/sprite-animations-v2.css` — append after the RANGED section (after line 363), before the NAVAL section

The pike thrust overrides the default `cq2-attack-swing` for melee pikes by:
1. Suppressing the outer `.cq-weapon` rotation (which is the default slash).
2. Animating `.cq-weapon-inner` (the inner `<g>` carrying the rest-pose transform) with a translate along local -Y (toward the tip).

The inner group's SVG attribute `transform="translate(54 22) rotate(-8)"` stays intact — CSS `transform` on the same element **adds** to the SVG attribute in browsers. The keyframe's `translate(0, -10px)` moves along the already-rotated local axis, so the thrust is shaft-aligned.

- [ ] **Step 1: Add the pike CSS section**

  Open `src/assets/sprite-animations-v2.css`. After the closing lines of the RANGED section (after `animation-delay: calc(var(--phase, 0) * -1.4s);` / `}` that ends the bowstring rule, around line 363), insert:

  ```css

  /* ─────────────────────────── PIKE THRUST ─────────────────────────── */

  /* Suppress the default overhead slash for pikes.
     The inner group carries the rest-pose SVG transform; animating it keeps
     the thrust shaft-local without baking the rest transform into keyframe stops. */
  .cq-v2[data-state="attack"][data-kind-variant="pike"] .cq-weapon {
    animation: none;
  }

  @keyframes cq2-attack-pike-thrust {
    0%   { transform: translate(0, 0); }
    18%  { transform: translate(0, 2px); }    /* pull-back — builds anticipation */
    32%  { transform: translate(0, -10px); }  /* thrust apex — tip extends forward */
    48%  { transform: translate(0, -10px); }  /* hold — the strike lands */
    72%  { transform: translate(0, -2px); }   /* recover — ease back */
    100% { transform: translate(0, 0); }
  }

  /* Animate the inner group, not .cq-weapon, so the rest-pose rotation
     (rotate(-8)) is already applied when CSS transform runs. */
  .cq-v2[data-state="attack"][data-kind-variant="pike"] .cq-weapon-inner {
    animation: cq2-attack-pike-thrust 1.4s cubic-bezier(.3,.7,.4,1) infinite;
    animation-delay: calc(var(--phase, 0) * -1.4s);
    transform-box: fill-box;
  }
  ```

  /* ─────────────────────────── HOUND (QUADRUPED) ─────────────────────────── */

  /* data-kind="hound" is the quadruped body-plan slot. Both ScoutHound and
     WarHound use it. Variant (scout / war) tunes tempo and range via CSS vars. */

  /* Idle: flat translateY-only breath — no scaleY (wrong silhouette for a dog) */
  @keyframes cq2-hound-breathe {
    0%, 100% { transform: translateY(0); }
    45%      { transform: translateY(-0.8px); }
    70%      { transform: translateY(-0.3px); }
  }
  .cq-v2[data-kind="hound"][data-state="idle"] .cq-sprite-figure {
    animation: cq2-hound-breathe 3.2s cubic-bezier(.4,0,.2,1) infinite;
    animation-delay: calc(var(--phase, 0) * -3.2s);
  }

  /* Walk body: two-peak bob (quadrupeds lift on each diagonal transition) */
  @keyframes cq2-hound-walk-body {
    0%, 100% { transform: translateY(0); }
    25%      { transform: translateY(var(--quad-bob, -1.5px)); }
    50%      { transform: translateY(0); }
    75%      { transform: translateY(var(--quad-bob, -1.5px)); }
  }
  .cq-v2[data-kind="hound"][data-state="walk"] .cq-sprite-figure {
    animation: cq2-hound-walk-body var(--quad-walk-cycle, 0.6s) ease-in-out infinite;
    animation-delay: calc(var(--phase, 0) * -1 * var(--quad-walk-cycle, 0.6s));
  }

  /* Leg pivot setup — apply to all four hooks; pivot at top of each leg group */
  .cq-v2[data-kind="hound"] .cq-leg-fl,
  .cq-v2[data-kind="hound"] .cq-leg-fr,
  .cq-v2[data-kind="hound"] .cq-leg-bl,
  .cq-v2[data-kind="hound"] .cq-leg-br {
    transform-box: fill-box;
    transform-origin: 50% 0%;
  }

  /* Two keyframes with opposite start positions = diagonal pair desync without
     extra animation-delay math. --quad-leg-scale lets WarHound trim the arc. */
  @keyframes cq2-hound-leg-a {
    0%, 100% { transform: rotate(calc(-10deg * var(--quad-leg-scale, 1))); }
    50%      { transform: rotate(calc(14deg * var(--quad-leg-scale, 1))); }
  }
  @keyframes cq2-hound-leg-b {
    0%, 100% { transform: rotate(calc(14deg * var(--quad-leg-scale, 1))); }

  }

  /* Diagonal pairs: FL+BR phase A, FR+BL phase B */
  .cq-v2[data-kind="hound"][data-state="walk"] .cq-leg-fl,
  .cq-v2[data-kind="hound"][data-state="walk"] .cq-leg-br {
    animation: cq2-hound-leg-a var(--quad-walk-cycle, 0.6s) cubic-bezier(.4,0,.6,1) infinite;
    animation-delay: calc(var(--phase, 0) * -1 * var(--quad-walk-cycle, 0.6s));
  }
  .cq-v2[data-kind="hound"][data-state="walk"] .cq-leg-fr,
  .cq-v2[data-kind="hound"][data-state="walk"] .cq-leg-bl {
    animation: cq2-hound-leg-b var(--quad-walk-cycle, 0.6s) cubic-bezier(.4,0,.6,1) infinite;
    animation-delay: calc(var(--phase, 0) * -1 * var(--quad-walk-cycle, 0.6s));
  }

  /* Attack body: forward lunge, no torso rotation (dog bites head-first) */
  @keyframes cq2-hound-attack-body {
    0%   { transform: translate(0, 0); }
    18%  { transform: translate(-3px, 2px); }                   /* crouch back */
    32%  { transform: translate(var(--quad-lunge, 8px), 0); }   /* lunge apex */
    48%  { transform: translate(var(--quad-lunge, 8px), 0); }   /* hold */
    72%  { transform: translate(1px, 1px); }                    /* recover */

  }

    animation: cq2-hound-attack-body var(--quad-attack-cycle, 1.4s) cubic-bezier(.3,.7,.4,1) infinite;
